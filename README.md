# ApexCare V2

Next-Gen Care Management System built with Next.js and Tailwind CSS.

## Features

- **Dashboard**: Central hub for managing care operations.
- **Service User Management**: Comprehensive profiles, medical history, and care plans.
- **Medication Tracker**: Track dosages, frequencies, and schedules.
- **Group Management**: Organize residents and staff into units or floors.
- **Admin Console**: Module management and system configuration.
- **Responsive Design**: Optimized for desktop and tablet usage.

## Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Database ORM**: [Prisma](https://www.prisma.io/) (SQLite)
- **Authentication**: [NextAuth.js](https://next-auth.js.org/) (Simulated for demo)
- **Icons**: [Lucide React](https://lucide.dev/)

## Getting Started

### Prerequisites

- Node.js 18+ installed.

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.
